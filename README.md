# Maestro-fe
메트로놈 프로젝트 "Maestro" 프론트 레포지토리입니다.


## Project Stack

**Front-end**

- React
- Styled Components
- React Router
- slick

## Branch Strategy

Github flow 전략을 사용합니다.

## Commit Message Convention

```
type: commit message
```

모든 커밋 메세지는 명령문 (현재시제) 를 사용하여 작성합니다. 모든 커밋 메세지 앞에는 다음의 `type` 을 작성합니다.

- `feat`: 새로운 기능 추가
- `fix`: 버그수정
- `docs`: 문서수정
- `style`: 코드 포매팅 등
- `refactor`: 코드 리팩토링
- `chore`: 기타 작업
